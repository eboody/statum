# statum-examples

`statum-examples` is a runnable examples crate for the Statum workspace.

## Run

```bash
cargo test -p statum-examples
cargo run -p statum --example 01-setup
```

## Contents

- Focused examples in `src/examples/`
- Integration-style scenario tests in `tests/examples.rs`, `tests/patterns.rs`, and `tests/stress.rs`
