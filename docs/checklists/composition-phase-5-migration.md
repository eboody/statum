# Composition Roadmap Checklist: Phase 5

Goal: downstream validation and migration.

- [ ] Pick at least one real consumer workspace as the migration proving
      ground.
- [ ] Convert the highest-value declared journeys into composition machines
      where the flow is truly protocol truth.
- [ ] Promote detached handoffs and opaque references only where exactness is
      honest.
- [ ] Verify the exact graph bundle shows more of the main workflow than the
      current journey-plus-heuristic story.
- [ ] Verify the inspector composition view is materially more useful than the
      old journey-first view.
- [ ] Write a migration guide from `journeys!`, `#[via(...)]`, and older
      attested helpers.
- [ ] Update examples to show the composition model in practice.

Acceptance gate:

- [ ] A real workspace can expose its main journey map through composition
      machines and exact handoffs, not just through declared journeys.
